# Contador Inteligente de Veículos com YOLOv8

- Este projeto utiliza **Visão Computacional** com o modelo **YOLOv8** para detectar, rastrear e contar veículos em tempo real usando a câmera no estacionamento. (No momento estamos rodando o projeto piloto focado na camera Durawell Full-Colorsmart Camera (Yoosee)  
- Registrar automaticamente a passagem de carros — e futuramente **identificar placas** e **registrar entradas de clientes** em um banco de dados SQL.


# Objetivo

Criar um sistema de **contagem e identificação de veículos** capaz de:
- Detectar e rastrear carros, caminhões e ônibus em tempo real;
- Contar o número de veículos que cruzam uma linha virtual;
- (Em desenvolvimento) **Ler e registrar placas** de veículos;
- (Em desenvolvimento) **Integrar com banco de dados SQL** para armazenar logs e gerar relatórios de movimentação.
